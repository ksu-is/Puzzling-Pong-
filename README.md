# Puzzling-Pong-
A gaming software that allows the user to hit a ball back and forth using PyGame Library. This is a single-player game that people of various ages can play. This two-dimensional game requires the use of a keyboard or mouse. The objective of this game is to ultimately get to ten points. Points are scored by getting the ball past your opponentâ€™s paddle. There are three different speeds which are easy, moderate, and difficult.
