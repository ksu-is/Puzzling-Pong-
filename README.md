# Puzzling-Pong-
A gaming software that allows the user to hit a ball back and forth using PyGame Library. This is a two- player game that people of various ages can play. This two-dimensional game requires the use of a keyboard or mouse. The objective of this game is to ultimately score the most points, you can set the points with your partner and manually close the game after the set point is reached.
